# Endless Runner Game Prototype

## Overview
This project is a fast-paced endless side-scrolling runner game where the player must time jumps over oncoming obstacles to avoid crashing. Through the development of this prototype, essential game development skills were acquired, including adding music and sound effects, creating dynamic endless backgrounds, and incorporating particle effects such as splatters and explosions.

## Features
- **Endless Side-Scrolling Gameplay:** A challenging and engaging game where the player must navigate obstacles.
- **Music and Sound Effects:** Background music and sound effects enhance the gaming experience.
- **Dynamic Endless Backgrounds:** Repeating backgrounds provide a seamless and continuous game environment.
- **Particle Effects:** Visual effects like splatters and explosions make the game more visually appealing and satisfying to play.

## Learning Objectives
By completing this project, the following skills were developed:
1. Implementation of endless side-scrolling mechanics in a game.
2. Addition and management of music and sound effects to enhance the player experience.
3. Creation of dynamic, repeating backgrounds essential for side-scrolling games.
4. Integration of particle effects to add visual flair and feedback to the game.
